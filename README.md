# Sprachwechsler MonkeyOffice Prosaldo

## MonkeyOffice
Wer mehrere Länder z.B. USA, UK etc. berechnen will hat immer wieder Probleme, die korrekten Formate in den Vorlagen zu erstellen. Manchmal muss man z.B. das Datum erst auseinander nehmen und dann wieder zusammensetzen. 

Da MonkeyOffice die Systemsprache nimmt, kommt die Datei Sprache.exe zum Tragen: Sie wechselt sehr schnell von DE auf UK, US und somit erkennt MonkeyOffice die neue Sprache. 

Die Datei kann auch im Betrieb von MonkeyOffice genutzt werden (MO offen) und bei dem Druck werden die Formate, wie Datum, Zahlen etc. korrekt angezeigt

## Betriebssystem
Windows11
Verwendet PowerShell (siehe Sprache.ps1). Die Datei Sprache.exe ist von mir signiert wurden. 

## Verwendung
In MonkeyOffice kann die exe oder ps1 Datei sofort ausgeführt werden, ohne den Computer neu zu starten oder den Benutzer zu wechseln- auch wenn MonkeyOffice offen ist, wird beim nächsten Druck eines Beleges die korrekte Formatierung angezeigt.
